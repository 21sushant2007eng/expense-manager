# expense-manager
My project is a simple Expense Manager application written in Python. It allows users to easily track their personal expenditures through a straightforward command-line interface (CLI). It stores each expense in a list as a dictionary containing the category and amount fields. The program continuously displays a menu where the user can choose different operations. When the user selects Add Expense, the program takes input for category and amount and appends it to the expense:list. The View History (Reversed) option displays all recorded expenses starting from the most recent entry by iterating the list in reverse order. The Calculate Total Expense option loops through all elements and sums the amounts to show the total spending. The Find Maximum Expense feature searches through the list to identify and display the expense with the highest amount. The menu is controlled using a while True loop and conditional statements to execute different functions based on user choice. The program continues running until the user selects the Exit option. This project demonstrates the use of lists, loops, functions, conditionals, and basic algorithms in Python.
1. Problem Definition:-
People often struggle to keep track of where their money is spent. This console-based Expense Manager provides a simple way to record daily expenses, view them later, calculate total spending, and identify the highest expense. It helps users maintain financial awareness without needing complex tools.
2. How to use:-
- Download the project files to your system.
- Open the project folder in a code editor or terminal.
- Run the program using
- Follow the menu displayed on the screen.
- Choose:-
- 1 to Add Expense
- 2 to View Expense History (Reversed)
- 3 to Calculate Total Expense
- 4 to Find Maximum Expense
- 5 to exit the porgram
- Enter the category and amount when prompted.
